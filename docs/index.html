Offline-first health tech

Clinical note prep for understanding and safe sharing.

NoteNest is a Clinical Note Prep Toolkit with two modes: Note Understanding (summarize, structure, and explain) and Safe Sharing (Cloak Mode de-identification). It protects sensitive details (names, emails, phone numbers, IDs) using placeholders while keeping clinical meaning intact. Designed for real workflows—referrals, second opinions, handoffs, and study—NoteNest helps patients understand their care and helps clinicians act faster, while keeping sensitive data protected.

Default style: Protected placeholders (e.g., [EMAIL_PROTECTED]).

Offline-first workflow

Use the CLI or the browser demo with zero paid APIs and deterministic output.

1. Ingest

Paste or upload a note. The CLI supports stdin, files, and batch folders.

2. Protect + summarize

Identifiers are masked before structured summaries are generated.

3. Share safely

Export patient and clinician views in txt, md, or json formats.

4. Stay local

No cloud calls required. Notes stay on-device for privacy and compliance.
